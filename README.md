# sxToys
Astronomy Image Capture Applications for Starlight Xpress (https://www.sxccd.com) cameras.

Using wxWidgets for cross-platform support. Use more, but smaller, applications for specific purposes (toys).

Time Delay Integration scanning (https://nexsci.caltech.edu/workshop/2005/presentations/Rabinowitz.pdf) is now somewhat functional. Here is a scan using an SXV-H9 camera and Meade 80mm (f/6) refractor (scroll down):
